====================================
프로젝트 손익계산기 - Apps Script 버전
====================================

[설치 방법]

1. Google 스프레드시트 생성
   - drive.google.com 접속
   - 새로 만들기 → Google 스프레드시트

2. Apps Script 열기
   - 메뉴: 확장 프로그램 → Apps Script

3. 파일 생성 및 코드 복사

   (1) Code.gs (기본 생성됨)
       - 기존 내용 삭제
       - Code.gs 파일 내용 전체 복사 붙여넣기

   (2) Index.html 생성
       - 좌측 파일 옆 + 버튼 → HTML
       - 이름: Index
       - Index.html 파일 내용 전체 복사 붙여넣기

   (3) Styles.html 생성
       - 좌측 파일 옆 + 버튼 → HTML
       - 이름: Styles
       - Styles.html 파일 내용 전체 복사 붙여넣기

   (4) JavaScript.html 생성
       - 좌측 파일 옆 + 버튼 → HTML
       - 이름: JavaScript
       - JavaScript.html 파일 내용 전체 복사 붙여넣기

4. 배포
   - 메뉴: 배포 → 새 배포
   - 유형 선택: 웹 앱
   - 설명: 프로젝트 손익계산기
   - 실행 사용자: 본인
   - 액세스 권한: 본인만 (또는 필요에 따라)
   - 배포 클릭

5. 사용
   - 배포 후 나오는 URL 클릭
   - 또는 스프레드시트에서: 확장 프로그램 → Apps Script → 배포 → 배포 관리 → URL 클릭


[기능 설명]

1. 프로젝트 정보 입력
   - 프로젝트명, 고객사, 견적가, 계약금액 등

2. 내부 인원 (월급 기반)
   - 직원 관리에서 직원 등록
   - 드롭다운에서 선택하여 추가
   - 투입시간 입력하면 자동 계산

3. 외부 인원 (일당 기반)
   - 구글시트에서 불러오기: 같은 스프레드시트의 다른 시트에서 불러옴
   - CSV 파일 불러오기: 컴퓨터의 CSV 파일 업로드
   - 직접 추가: 수동 입력

4. 기타 비용
   - 전기 자재비, 출장 경비, 외주 가공비, 운반/포장비, 소모품비

5. 간접비 및 마진
   - 공통 관리비율, 하자보수 예비비율, 목표 마진율

6. 견적 배분
   - 계약금액을 항목별로 배분

7. 대시보드
   - 손익 현황: 비용 항목별 합계, 총 비용, 손익
   - 비용 비교: 배분 vs 실제 비교


[외부 인원 공수표 시트 형식]

열 구조:
A: 이름
B: 업체명
C: 직급
D: 일당
E: 투입일수
F: 비용항목 (배선공사/배관공사/설계/판넬/셋업/기타)

1행은 헤더, 2행부터 데이터


[데이터 저장 위치]

- 프로젝트 데이터: "프로젝트데이터" 시트 (자동 생성)
- 직원 마스터: "직원마스터" 시트 (자동 생성)
