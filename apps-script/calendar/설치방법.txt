====================================
 달력 출근부 시스템 - 설치 방법
====================================

[ 기능 설명 ]
- 달력 시트에서 날짜 셀 클릭 후 사이드바에서 출근자 선택
- 시트1의 A열에 있는 이름 목록을 체크박스로 표시
- 선택한 이름들이 해당 날짜 셀에 자동 입력


[ 설치 순서 ]

1. 구글 스프레드시트 열기
   - https://docs.google.com/spreadsheets/d/1nLC-5mNclNLg_94Ond0zJSalHt4qNCzwWy1tAAEV8UY 열기

2. Apps Script 편집기 열기
   - 메뉴: 확장 프로그램 > Apps Script 클릭

3. 기본 코드 파일 수정
   - 왼쪽에 "Code.gs" 파일이 보임
   - 기존 내용을 모두 지우고
   - CalendarCode.gs 파일 내용을 전체 복사해서 붙여넣기

4. HTML 파일 추가
   - 왼쪽 "파일" 옆의 + 버튼 클릭
   - "HTML" 선택
   - 파일명: CalendarSidebar (자동으로 .html이 붙음)
   - CalendarSidebar.html 파일 내용을 전체 복사해서 붙여넣기

5. 저장 및 권한 부여
   - Ctrl+S 또는 저장 버튼 클릭
   - 처음 실행 시 권한 승인 필요:
     * "권한 검토" 클릭
     * 구글 계정 선택
     * "고급" 클릭 > "프로젝트명(안전하지 않음)으로 이동" 클릭
     * "허용" 클릭

6. 스프레드시트 새로고침
   - 브라우저에서 스프레드시트 새로고침 (F5)
   - 잠시 후 메뉴에 "출근부"가 나타남


[ 사용 방법 ]

1. 시트1 준비
   - 시트1의 A열에 직원 이름 목록 입력
   - 예: A1에 "홍길동", A2에 "김철수"...

2. 달력 시트 생성 (선택사항)
   - 메뉴: 출근부 > 달력 시트 생성
   - 년월 입력 (예: 2025-01)
   - 자동으로 달력 형태의 시트가 생성됨

3. 출근자 선택
   - 달력에서 원하는 날짜 셀 클릭
   - 메뉴: 출근부 > 출근자 선택
   - 사이드바에서 출근한 사람 체크
   - "저장" 버튼 클릭
   - 선택한 이름들이 셀에 "홍길동, 김철수" 형태로 입력됨


[ 주의사항 ]
- 시트1의 A열에 이름이 있어야 함
- 셀을 먼저 선택한 후 사이드바를 열어야 함
- 셀을 바꾸면 "셀 정보 새로고침" 버튼 클릭 필요


[ 문제 해결 ]
- "출근부" 메뉴가 안 보이면: 스프레드시트 새로고침
- 권한 오류 시: Apps Script에서 직접 onOpen 함수 실행
- 이름이 안 보이면: 시트1의 A열에 이름이 있는지 확인
